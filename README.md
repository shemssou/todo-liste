# 📝 To-Do Liste mit JavaScript

Dieses Projekt ist eine einfache, interaktive Aufgabenliste, die mit HTML, CSS und JavaScript erstellt wurde.

## 🔧 Funktionen:
- Aufgaben hinzufügen
- Aufgaben als erledigt markieren (durchstreichen)
- Aufgaben löschen

## 💻 Technologien:
- HTML
- CSS
- JavaScript (Vanilla)
